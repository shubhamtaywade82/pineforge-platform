# Primer: First Steps

Source: https://www.tradingview.com/pine-script-docs/primer/first-steps/

## Learning path

1. Use existing scripts on TradingView.
2. Read Pine Script source from open-source publications.
3. Write your own indicators, strategies, or libraries.

## Script types in Pine Script v6

- **Indicators** — calculations and plots (e.g. RSI, MACD).
- **Strategies** — include `strategy.*()` order logic; backtestable in Strategy Tester.
- **Libraries** — reusable functions imported by other scripts.

## Using existing scripts

Load from the chart via **Indicators, metrics, and strategies** (`/` shortcut). Categories include Favorites, Personal (Pine Editor saves), Technicals (built-ins), Community (150k+ scripts), and Invite-only.

Open-source Community scripts expose full source on the script page. Built-in indicators may expose source via the `{}` icon in the chart legend (Create working copy to edit).

## Reading code

Best sources for learning:

- TradingView built-in indicators
- Editors' Picks (open-source)
- Reputable open-source authors

## Writing scripts

Pine Script v6 targets both new and experienced programmers. Start with a first indicator, then explore the User Manual and Reference Manual.
